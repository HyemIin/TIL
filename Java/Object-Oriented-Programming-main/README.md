## 요구사항
- James와 Tomas는 각각 버스와 지하철을 타고 학교를 간다.
- James는 5000원을 갖고 있다.
- James는 100번 버스를 타면서 비용으로 1000원을 지불한다.
- Tomas는 10000원을 갖고 있다.
- Tomas는 지하철을 타면서 1200원을 지불한다.

### 추가 요구사항
- Edward는 택시를 타고 학교를 간다.
- Edward는 20000원을 갖고 있다.
- Edward는 택시를 타면서 10000원을 지불한다.
- 택시의 회사명은 "잘나간다 운수" 이다.
